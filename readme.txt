Hola Mundo en GitHub Actions
Enunciado: Crea un flujo de trabajo sencillo que se ejecute automáticamente cada vez que haya un push a la rama principal del repositorio.
En este workflow, configura un job que imprima por pantalla un mensaje "Hola Mundo" (usando un comando echo) para verificar que todo está funcionando. 
Este ejercicio te ayudará a montar la estructura básica de un workflow y a familiarizarte con la sintaxis YAML mínima necesaria.
